A Dart FFI binding for libgit2

## Features

* [x] get libgit2 version
* [ ] create local repo 
* [ ] clone local repo
* [ ] clone remote repo
* [ ] browse history
* [ ] show commits from history
* [ ] get/set user name, email
* [ ] stage/unstage files
* [ ] commit
* [ ] stashing
